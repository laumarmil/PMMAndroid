Introduccion a Room
Base de datos Android

Arquitectura: Activity--->ViewModel[liveData]--->Repositorio--->RoomDatabase[DAO]-->[SQLite]

Implementacion de Room
Para usar esta librería seguiremos los siguientes pasos:

    Añadir dependencia de Room a Gradle
    Crear una Entity por cada modelo
    Crear DAOs
    Crear subclase de RoomDatabase
    Crear Repositorio
    Testear capa de datos
    Testear Migraciones

Vamos a trabajar:
CRUDs[create read update y delete]
Relaciones (entre tablas)
Busquedas
Visitas
